🚀 WordPress 사이트 자동 구축

사이트 정보를 입력하고 개인 도메인과 Cloudflare API를 연결하면 WordPress가 자동으로 설치됩니다.

  • 개인 도메인(예: myblog.com)으로 완전 운영 — 서브도메인 노출 없음
  • Cloudflare Worker 자동 배포 — 도메인 프록시 무중단
  • WordPress 최신버전 자동 설치 (한국어, PHP 8.3)
  • Cron Job 강제 활성화 · 어드민/모든 페이지 개인도메인으로 제공
  • 글·페이지·미디어 라이브러리 → Cloudflare D1 + KV 자동 연동
  • 속도 최적화 (gzip, 브라우저 캐시, Breeze)
결제 완료된 플랜과 동일해야 생성됩니다.

개인 도메인 설정
사용자에게 보이는 도메인입니다.
예: myblog.com 또는 www.myblog.com
실제 WordPress 서버(호스팅 서브도메인)는 숨겨지고, 모든 페이지·관리자 페이지·크론잡이 이 도메인으로 동작합니다.
http(s):// 없이 도메인만 입력 (예: myblog.com)
☁️ Cloudflare Worker가 자동으로 연결됩니다.
별도의 API 키 입력 없이 서버 관리자가 미리 설정한 Worker가 개인 도메인과 자동으로 연결됩니다.
도메인 DNS에서 CNAME 레코드만 Worker 주소로 지정하면 완료됩니다.

인프라 구성 중...

Cloudflare Worker · D1 · KV 설정 및 WordPress 파일 업로드 중입니다.
⏱️ 예상 시간: 13~15분 — 안정적인 설치를 위해 충분한 시간을 사용합니다. 다른 탭으로 이동해도 계속 진행됩니다.

경과 시간: 00:00 | 예상: 13~15분
⏳ 1단계: D1 데이터베이스 + KV 네임스페이스 생성
⏳ 2단계: WordPress D1 스키마 초기화 (테이블·관리자 계정)
⏳ 3단계: WordPress 파일 KV 업로드 (CSS·JS·테마 등, 13~15분)
⏳ 4단계: Worker 업로드 + IP 방식 도메인 연결 (A 레코드)
WordPress 파일 업로드 중... (약 13~15분 소요, 다른 탭으로 이동해도 계속 진행됩니다)
