WordPress 사이트 자동 구축

사이트 정보를 입력하면 Cloudflare 인프라 위에 WordPress가 자동으로 설치됩니다.

  • 개인 도메인(예: myblog.com)으로 완전 운영 — 서브도메인 노출 없음
  • Cloudflare Worker 자동 배포 — 도메인 프록시 무중단
  • WordPress 최신버전 자동 설치 (한국어, PHP 8.3)
  • D1 데이터베이스 + KV 스토리지 자동 연동
  • 속도 최적화 (gzip, 브라우저 캐시 자동 적용)
관리자/매니저는 어떤 플랜이든 제한 없이 생성 가능합니다.

도메인 설정

개인 도메인을 입력하세요. DNS는 자동으로 설정됩니다.

개인 도메인 설정
www 없이 입력 (예: myblog.com). Cloudflare 계정에 등록된 도메인이어야 합니다.
자동 DNS 설정 안내: Cloudflare API를 이용해 A 레코드, Worker Route가 자동으로 등록됩니다. 도메인이 Cloudflare에 등록되어 있어야 하며, DNS 전파까지 최대 48시간이 소요될 수 있습니다.
